Table Tags

This table will explain how to create tables in html and how to fill them with data

Tag Name Description
< table > Table The wrapper element of all html tables
< Thead > > Table head The set of rows defining the column headers in a table
< tbody > Table body The set of rows containing the actual table data
< tr > Table row The table row container
< td > Table data The cell content
< tfoot > Table Foot The set of rows defining the footer in a table

Explainer

  • Tables are opened wit a < table > everything that follows is nested within this table element
  • The table head is created with < thead > and closed before the body starts
  • Next comes < tbody > which indicates the table body
  • Within the body there are several rows < tr > each row needs to be filled with data < td >. Each row contains several tds
